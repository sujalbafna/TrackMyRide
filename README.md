
# MIT ADT - Track My Ride

Welcome to the **Track My Ride** project, developed for MIT Art, Design & Technology University. This is a comprehensive transportation management system.

## 🚀 Quick Start
1. **Access the App**: Run `npm run dev` to start the local environment.
2. **Select a Role**: Choose between Admin, Student, Faculty, Driver, or Bus Terminal on the landing page.
3. **Register**: New accounts (except Admin) require administrative approval before login is enabled.

## 🛠 Features
- **Real-Time Tracking**: road-snapped bus movement using Firestore & Google Maps.
- **Navigation Overlay**: Professional driving instructions for bus operators.
- **Code Attendance**: Secure 4-digit daily verification system.
- **SOS Security**: Instant emergency alerts for passengers and drivers.
- **Admin Analytics**: Fleet occupancy and attendance reports.
